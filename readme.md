## Getting Started

## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:shavinski/express-jobly.git
   ```
2. Install dependencies
   ```sh
   npm install 
   ```
3. Start the local server
   ```sh
   node server.js
   ```
